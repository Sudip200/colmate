# colmate
This is a Dating App for connecting like minded people from different colleges 

# Features Includes

- Swipe Left & Right to reject and accept 
- Live chat after getting sucessfull matches 

# Proposed Features 

- Complex feed algorithm
- Live Video Call
- UI enhancements
